# Netflix Movies and TV Shows
## Source Data
https://www.kaggle.com/datasets/shivamb/netflix-shows

## Purpose
To provide a breakdown on Netflix content by Movies/TV, directors, content ratings, etc.

## Tools
- SQL (Postgres)
- Python (pandas, seaborn, Matplotlib)
